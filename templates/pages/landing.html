How ASR Gateway works

1) Upload

Humans use JWT; applications use API tokens. Both send audio to the same gateway.

2) Process

Jobs are queued, processed by the ASR core, and metered against plans.

3) Notify

Users see live status in the dashboard and can retrieve transcripts instantly.
